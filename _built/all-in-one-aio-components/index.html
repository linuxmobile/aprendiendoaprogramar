    Inside a list

    1. Calculate the Standard Deviation (SDSDSD) of the current node (let's say SSS, parent node) by using MSE or MAE,

      SD(S)=1n∑i=1n(yi−yˉi)2,or SD(S)=1n∑i=1n∣yi−yˉi∣,\begin{aligned}SD(S) &= \frac{1}{n} \sum_{i=1}^{n} (y_i - \bar{y}_i)^2, \\\text{or } SD(S) &= \frac{1}{n}\sum_{i=1}^n \vert y_i - \bar{y}_i \vert,\end{aligned}SD(S)or SD(S)​=n1​i=1∑n​(yi​−yˉ​i​)2,=n1​i=1∑n​∣yi​−yˉ​i​∣,​

      where yi∈y_i\inyi​∈ the target values (Hours Played in the above example), yˉ=Σyn\bar{y}=\frac{\Sigma y}{n}yˉ​=nΣy​ is the mean value and nnn is the number of examples in this node.

    2. Calculate the Standard Deviation Reduction (SDR) after splitting node SSS on each attribute (for example, consider attribute OOO). The attribute w.r.t. the biggest SDR will be chosen!

      SDR(S,O)⏟Standard Deviation Reduction=SD(S)⏟SD before split−∑jP(Oj∣S)×SD(S,Oj)⏟weighted SD after split\underbrace{SDR(S,O)}_{\text{Standard Deviation Reduction}}= \underbrace{SD(S)}_{\text{SD before split}}- \underbrace{\sum_j P(O_j | S) \times SD(S,O_j)}_{\text{weighted SD after split}}Standard Deviation ReductionSDR(S,O)​​=SD before splitSD(S)​​−weighted SD after splitj∑​P(Oj​∣S)×SD(S,Oj​)​​

      where j∈j \inj∈ number of different properties in OOO and P(Oj)P(O_j)P(Oj​) is the propability of property OjO_jOj​ in OOO. Note that, SD(S,Oj)SD(S,O_j)SD(S,Oj​) means the SD of node OjO_jOj​ which is also a child of node SSS.

    3. After splitting, we have new child nodes. Each of them becomes a new parent node in the next step. Go back to step 1
