    This is my personal list of to-do things after reinstall Windows.

    ðŸ‘‰ Note: Ubuntu / Pop!_OS fresh start.
    ðŸ‘‰ Note: Mac fresh installation.

    1. Update and install drivers.
    2. Google Chrome browsers and sign in to sync. Install also these extensions: mate translate, google dictionary, TabCloud, raindrop, last pass bitwarden, AVIM, adblock, GNOME Shell integration.
    3. Bitwadern. In case you wanna enable "Unlock with Windows Hello" (if it doesn't appear normally), you have to install the latest supported Visual C++ downloads.
      1. Setting up: lock when system lock and tick on option "Enable browser integration" (no need to tick on "Require verification...").
    4. (Optional) DeepL: best translator for Windows using AI.
    5. (Optional) QTTranslate: quick and easy translate words and hear the pronunciation.
    6. Unikey: Vietnamese Input Method.
    7. Log in to online accounts (Google, Stack Overflow, Github, Dropbox,...)
    8. cmder: drop-down terminal & install personal setting file (can be found in /config/) ðŸ‘‰ Check this note to set up WSL2 and Bash.
    9. IObit Uninstaller.
    10. WinRAR.
    11. (Optional) Notion + Ticktick.
    12. Git & Gitkraken (recommended, check this note) / Github Desktop.
      1. Clone repositories from Github.
    13. Visual Studio Code and extensions (Bracket Pair Colorizer, Docker, Markdown All in One, Markdown Shortcuts, Python, Remote Development, Auto Close Tag, ...). Normally, install once, then sync with Github / Microsoft account.
    14. (Optional) Microsoft Office. Could use the free version (on web).
    15. GoldenDict + dictionaries & Lingoes + dictionaries.
    16. Uninstall OneDrive.
    17. (Optional) Microsoft Teams + Slack + Trello.
    18. LockHunter: delete being used files/folders.
    19. QTTabBar: Tabs for Windows Explorer. After installing and opening, choose View > Options > tick on QTTabBar only then right click on toolbar and choose Lock the toolbar.
    20. (Optional) MailSpring: email client. I use Mailbird (paid).
    21. Mail and Calendar but don't enable sync for Mail.
    22. (Optional) TeXLive + Texmaker.
    23. Google Backup and Sync.
    24. Anaconda (python and it dependencies)
    25. Docker Desktop.
    26. Internet Download Manager (paid) / Extreme Download Manager (free).
    27. (Optional) Matlab.
    28. (Optional) Affinity Designer (paid, alternative to Adobe Illustrator).
    29. K-Lite Codec Pack: video media player.
    30. Foxit Reader: PDF reader with tabs.
    31. Custom/Disable fn function on keyboard K380 -> download Logitech Options -> install & signin with dinhanhthimail@gmail.com -> check on "Use F1-F12 as standard function keys".
